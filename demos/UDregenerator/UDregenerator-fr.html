Affichage du graphe/arbre de Universal Dependencies avec re-génération avec jsRealB

[English version]
  • Copier une ou plusieurs structures de dépendances en format CONLLU
  • Cliquer sur le bouton Analyser pour créer le menu. Si Que les différentes est coché, seules les phrases dont la regénération est différente du texte original apparaissent dans le menu.
  • Choisir une phrase du menu. Une phrase débutant par une astérisque indique une structure de dépendance non projective.
  • Le graphe ou l'arbre de dépendances sera affiché. Le mot racine est affiché en rouge. En plaçant le curseur sur un mot, les informations sur le mot sont affichées dans une info-bulle.
  • Une phrase en français sera regénérée en utilisant jsRealB
  • La table indique les opérations d'édition pour transformer la phrase regénérée en la phrase originale: remplacement, insertion, délétion. Les différences d'espace sont ignorées.
  • L'expression jsRealB apparaît dans la fenêtre d'édition qui permet de la modifier et de réaliser l'expression modifiée en cliquant sur le bouton Réaliser
